# 搜索工具更新说明

## 更新日期
2026-03-11

## 更新内容

将项目中的文件搜索工具从 `Everything` 框架替换为 `ripgrep` 框架，提供更高效、更灵活的文件搜索功能。

## 主要变更

### 1. 新增模块
- **src/search_tools.py**: 新增独立的搜索工具模块，提供以下功能：
  - `search_files_by_name()`: 按文件名搜索文件
  - `search_content()`: 按内容搜索文件（搜索文件内部的文本）
  - `count_matches()`: 统计匹配数量
  - 底层函数：`ripgrep_search()`, `ripgrep_files()`, `ripgrep_count()`

### 2. 更新模块
- **src/windows_tools.py**:
  - 添加了 `RIPGREP_AVAILABLE` 标志来检查 ripgrep 是否可用
  - 更新了 `search_files_tool()` 函数，现在默认使用 ripgrep
  - 新增了 `search_content_tool()` 函数，用于按内容搜索
  - 更新了工具定义 `search_files` 的描述
  - 新增了 `search_content` 工具定义
  - 更新了 `TOOL_FUNCTIONS` 映射，添加了 `search_content`

## 功能对比

| 功能 | Everything | Ripgrep | 说明 |
|------|------------|---------|------|
| 按文件名搜索 | ✅ | ✅ | 支持通配符和正则表达式 |
| 按内容搜索 | ❌ | ✅ | 新增功能 |
| 统计匹配 | ✅ | ✅ | |
| 大小过滤 | ✅ | ⚠️ | 需要使用其他工具配合 |
| 时间过滤 | ✅ | ⚠️ | 需要使用其他工具配合 |
| 性能 | 极快 | 很快 | Everything 需要后台服务 |
| 依赖 | Everything 服务 | rg 命令行工具 | 更轻量 |

## 使用方法

### 按文件名搜索

```python
# 搜索所有 Python 文件
search_files_tool(pattern="*.py", directory="D:\\projects")

# 搜索包含特定关键词的文件
search_files_tool(pattern="config", directory="D:\\projects", max_results=100)

# 使用正则表达式
search_files_tool(pattern=".*\\.(py|txt)$", directory="D:\\projects")
```

### 按内容搜索（新增）

```python
# 搜索包含特定文本的文件
search_content_tool(pattern="TODO", directory="D:\\projects")

# 忽略大小写
search_content_tool(pattern="import", directory="D:\\projects", ignore_case=True)

# 限制文件类型
search_content_tool(pattern="def main", directory="D:\\projects", file_type="py")

# 显示上下文
search_content_tool(pattern="error", directory="D:\\logs", show_context=2)
```

### AI 工具调用示例

```json
// 按文件名搜索
{
  "name": "search_files",
  "arguments": {
    "pattern": "*.pdf",
    "directory": "D:\\documents",
    "max_results": 50
  }
}

// 按内容搜索
{
  "name": "search_content",
  "arguments": {
    "pattern": "API_KEY",
    "directory": "D:\\projects",
    "file_type": "py",
    "ignore_case": true,
    "max_results": 20,
    "show_context": 1
  }
}
```

## 安装要求

### 安装 Ripgrep

需要先安装 ripgrep 命令行工具：

**Windows (推荐):**
- 使用 Scoop: `scoop install ripgrep`
- 使用 Chocolatey: `choco install ripgrep`
- 手动下载: 从 [GitHub Releases](https://github.com/BurntSushi/ripgrep/releases) 下载并添加到 PATH

**验证安装:**
```bash
rg --version
```

### Python 依赖

项目不需要额外的 Python 包，直接调用 `rg` 命令行工具。

## 兼容性

- **向后兼容**: 保留了 Everything 框架的导入，但默认不再使用
- **回退机制**: 如果 ripgrep 不可用，会自动回退到 os.walk 递归搜索
- **工具名称**: 保持 `search_files` 工具名称不变，确保现有代码正常工作

## 性能优势

1. **无需后台服务**: Everything 需要运行后台服务，ripgrep 是纯命令行工具
2. **跨平台**: ripgrep 可以在 Windows、Linux、macOS 上使用
3. **轻量级**: 无需安装大型软件，只需一个可执行文件
4. **内容搜索**: 支持按文件内容搜索，这是 Everything 不支持的功能

## 注意事项

1. **路径问题**: ripgrep 默认搜索当前目录，使用绝对路径时需要确保路径正确
2. **权限问题**: 搜索系统目录可能需要管理员权限
3. **性能优化**: 建议使用 `glob` 参数限制搜索范围，提高搜索速度
4. **编码问题**: ripgrep 默认使用 UTF-8 编码，非 UTF-8 文件可能无法正确搜索
