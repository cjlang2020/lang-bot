# QQ机器人发送图片/文件说明

## 核心流程

```
1. 调用 post_c2c_file() 上传资源 → 获得 Media 对象
2. 调用 post_c2c_message() 发送消息（msg_type=7，带上 media）
```

## 私聊发送示例代码

```python
# -*- coding: utf-8 -*-
import botpy
from botpy.message import C2CMessage

class MyClient(botpy.Client):
    async def on_c2c_message_create(self, message: C2CMessage):
        # 发送图片示例
        image_url = "https://example.com/image.jpg"  # 图片URL
        
        # 步骤1: 上传资源
        media = await message._api.post_c2c_file(
            openid=message.author.user_openid,
            file_type=1,  # 1=图片, 2=视频, 3=语音
            url=image_url
        )
        
        # 步骤2: 发送消息
        await message._api.post_c2c_message(
            openid=message.author.user_openid,
            msg_type=7,  # 7=富媒体消息
            msg_id=message.id,
            media=media
        )

if __name__ == "__main__":
    intents = botpy.Intents(public_messages=True)
    client = MyClient(intents=intents)
    client.run(appid="your_appid", secret="your_secret")
```

## file_type 参数说明

| 值 | 类型 | 格式 |
|---|---|---|
| 1 | 图片 | png/jpg |
| 2 | 视频 | mp4 |
| 3 | 语音 | silk |

## API 方法说明

### post_c2c_file() - 上传资源

```python
await message._api.post_c2c_file(
    openid: str,          # 用户openid
    file_type: int,       # 媒体类型：1图片, 2视频, 3语音
    url: str,             # 资源URL
    srv_send_msg: bool    # 是否直接发送（默认False）
) -> message.Media        # 返回Media对象
```

### post_c2c_message() - 发送消息

```python
await message._api.post_c2c_message(
    openid: str,                    # 用户openid
    msg_type: int = 0,              # 消息类型：0文本, 7富媒体
    content: str = None,            # 文本内容
    media: message.Media = None,    # 富媒体对象
    msg_id: str = None,             # 回复的消息ID
    msg_seq: int = 1,               # 消息序号
) -> message.Message
```

## msg_type 参数说明

| 值 | 类型 |
|---|---|
| 0 | 文本消息 |
| 1 | 图文混排 |
| 2 | markdown |
| 3 | ark |
| 4 | embed |
| 7 | 富媒体（图片/视频/语音） |

## 注意事项

1. **资源URL要求**：资源需要是**可访问的URL**，不支持本地文件直接发送
2. **被动回复有效期**：5分钟
3. **消息类型**：发送图片/视频/语音时，msg_type 必须设置为 7

## 群聊发送

群聊发送类似，使用以下方法：

```python
# 上传群资源
media = await message._api.post_group_file(
    group_openid=group_openid,
    file_type=1,
    url=image_url
)

# 发送群消息
await message._api.post_group_message(
    group_openid=group_openid,
    msg_type=7,
    msg_id=message.id,
    media=media
)
```
