# 快速修复

修复了 `src/windows_tools.py` 文件中的错误：将 `false` 改为 `False`（Python 的布尔值应该使用首字母大写）。

## 错误原因

在 Python 中，布尔值应该写成 `True` 或 `False`，而不是 JSON 中的小写 `true` 或 `false`。

## 修复内容

修改了 `search_content` 工具定义中的 `ignore_case` 参数默认值：
```python
# 错误写法（JSON风格）
"ignore_case": {"type": "boolean", "description": "是否忽略大小写", "default": false}

# 正确写法（Python风格）
"ignore_case": {"type": "boolean", "description": "是否忽略大小写", "default": False}
```

## 验证

现在应该可以正常启动了：
```bash
python start_listener.py
```
